# Student Management System (Java)

A console-based student management application built with Java.

## Features

- Add new students
- List all students
- Search students by ID or name
- Remove students
- Calculate class average
- Persistent storage using CSV file

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File handling (CSV)
- Console application

## How to Run

Compile:
```
javac -d out *.java
```

Run:
```
java -cp out Main
```

Example usage:
```
=== Student Management ===
1) Add student
2) List students
3) Search (name or ID)
4) Remove by ID
5) Class average
0) Exit
```

Data is saved automatically in:
```
students.csv
```
